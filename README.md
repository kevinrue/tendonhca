## Data sets

`single nuc` refers to pools of cells from patients A-H.

## Subdirectories

`single_nuc_cellranger/` contains the output of [snakemake-workflows/cellranger-multi](https://snakemake.github.io/snakemake-workflow-catalog/docs/workflows/snakemake-workflows/cellranger-multi) for the single nuc data.
